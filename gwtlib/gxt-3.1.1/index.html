Welcome to GXT 3.1.1

GXT 3.1 is a pure GWT application framework that works everywhere from IE8 to the latest Firefox and Chrome builds. It enables you to create the best cross-platform applications using nothing but a browser, and has a phenomenal API.

Our latest release comes with a new theme, Neptune, and the tools we used to make it so you can generate your own from just a config file.
